## Status

> **Planned** — This repository is part of the AI Weekly platform roadmap. Development has not yet started.

## Planned Features

| Feature | Description |
|---------|-------------|
| Article Reader | Clean, responsive reading experience |
| Category Browsing | Filter by 10 AI domains |
| Search | Full-text search with filters |
| Multi-language | i18n with locale routing |
| SEO | Server-side rendering for discoverability |
| RSS Feed | Public RSS output for subscribers |
| Authentication | Login/signup with subscription management |

## Planned Tech Stack

| Component | Technology |
|-----------|-----------|
| Framework | Next.js 15 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| State | React Server Components + SWR |
| Auth | NextAuth.js |
| Deployment | Vercel |
